# 🎓 MIT Polytechnic College Website

## 📌 Project Overview

This project is a **responsive college website** developed as part of the Web Page Designing subject for Diploma in Artificial Intelligence & Machine Learning (AIML).

The website provides basic information about the college such as courses, admissions, and contact details in a clean and minimal design.

---

## 🏫 College Details

* **College Name:** MIT Polytechnic
* **Location:** Chhatrapati Sambhajinagar, Maharashtra, India
* **Course:** Diploma in AIML
* **Semester:** First Year, Second Semester

---

## 💻 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🌐 Features

* Responsive design (mobile & desktop friendly)
* Clean and minimal user interface
* Multiple pages:

  * Home
  * About
  * Courses
  * Admissions
  * Contact
* Basic form validation using JavaScript
* Simple navigation bar and footer
